# KHStrategyLab Thin MA Signal Source Notes

## 핵심 방향

Thin 기준의 TOP20 실시간 0B 보정 리소스는 줄이고, 추적중인 매수후보의 10분봉 MA5/MA20/MA60 신호등에 리소스를 배정한다.

## 화면 표시

`GridLeading`에 다음 컬럼을 런타임으로 추가한다.

- 10분5
- 10분20
- 10분60
- 신호

## 신호등 기준

- `5 > 20 > 60`: 🟢🟢 강
- `5 > 20 && 5 > 60`: 🟢 가능
- `60 > 5 && 60 > 20`: 🔴 약
- 그 외: 🟡 관망

## 데이터 기준

- 초기: REST 분봉 조회로 10분봉 최소 60개, 5분봉 최소 20개 확보
- 장중 표시: 0B 현재가를 현재봉 close/high/low에 반영
- KRX/NXT: `StrategyMarket` 기준으로 분봉 요청 코드 분리

## 전략 판단

화면 신호등은 현재봉 포함이다.
전략 매수신호는 마지막 확정 10분봉 기준으로 녹색 신호를 확인하고, 5분봉 20봉 신고가와 전일종가 기준가 위 조건을 함께 확인한다.

## 남은 확인 포인트

NXT 분봉 요청 코드 `_NX`, `_AL`, 6자리 fallback 중 실제 공식 권장값은 가이드AI 답변과 실전 로그로 확인해야 한다.
