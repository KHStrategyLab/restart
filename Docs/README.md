# KHStrategyLab Docs

현재 기준 문서는 아래 4개를 우선 본다.

```text
KHStrategyLab_User_Manual.md
KHStrategyLab_Architecture_Guide.md
KHStrategyLab_Strategy_Manual.md
KHStrategyLab_Update_Log.md
```

문서 역할:

```text
User Manual        실행/설정/화면 사용 기준
Architecture Guide 개발자용 프로그램 구조
Strategy Manual    전략 판단 기준
Update Log         이후 변경 기록
```

기존 개발 중 기록과 과거 설계 메모는 `Archive/` 폴더에 보관한다.
